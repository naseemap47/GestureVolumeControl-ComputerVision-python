## Description
Creating a Gesture volume control using OpenCV and Mediapipe in python

## Requirements
#### Programming Language
* Python3 (pyCharm)
#### Libraries
* OpenCV
* Mediapipe
* time
* math
* Numpy
* To change system Volume
  * subprocess (Linux)
  * pycaw (Windows)
